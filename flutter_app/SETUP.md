# Flutter Project Setup

## Quick Start

1. **Create Flutter project structure:**
   ```bash
   flutter create --org app.timetrade timetrade_wallet
   cd timetrade_wallet
   ```

2. **Copy the Dart code:**
   ```bash
   # Replace the lib/ folder with flutter_app/ contents
   rm -rf lib/*
   cp -r /path/to/flutter_app/* lib/
   mv lib/pubspec.yaml .
   mv lib/analysis_options.yaml .
   ```

3. **Create assets folder:**
   ```bash
   mkdir -p assets/images
   ```

4. **Install dependencies:**
   ```bash
   flutter pub get
   ```

5. **Run the app:**
   ```bash
   # iOS (requires Mac with Xcode)
   flutter run -d ios
   
   # Android (requires Android Studio)
   flutter run -d android
   ```

## iOS Setup

For iOS, add these to `ios/Runner/Info.plist`:

```xml
<!-- Face ID / Touch ID -->
<key>NSFaceIDUsageDescription</key>
<string>Enable Face ID to unlock your wallet</string>

<!-- Camera for QR scanning -->
<key>NSCameraUsageDescription</key>
<string>Camera is used to scan QR codes</string>
```

## Android Setup

For Android, add to `android/app/src/main/AndroidManifest.xml`:

```xml
<uses-permission android:name="android.permission.USE_BIOMETRIC"/>
<uses-permission android:name="android.permission.INTERNET"/>
<uses-permission android:name="android.permission.CAMERA"/>
```

Update `android/app/build.gradle`:
```gradle
android {
    defaultConfig {
        minSdkVersion 23  // Required for biometrics
    }
}
```
